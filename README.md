# PH-BRAC

This repository is being prepared for the PH-BRAC study.

Anonymized feature-level data, split files, and reproduction scripts will be released after institutional review and before formal publication.

## Repository Status

This is the phase 1 repository setup. The current files provide the public structure for the study repository while data release and reproduction materials are under institutional review.

## Planned Release Contents

- Anonymized feature-level data used for model development and evaluation
- Split files for training, validation, and testing
- Reproduction scripts for preprocessing, model training, evaluation, and figure/table generation
- Documentation describing variables, file formats, and reproduction steps

## Current Contents

- `docs/data_description.md`: planned data scope and release notes
- `docs/reproduction_plan.md`: planned reproduction workflow
- `requirements.txt`: provisional Python package requirements
- `LICENSE`: repository license for released code and documentation

## Data Availability

No personally identifiable student data are included in this repository. Future releases will contain anonymized feature-level data only after institutional review.

## Reproduction

The full reproduction workflow is not included in this phase. See `docs/reproduction_plan.md` for the planned release sequence.
