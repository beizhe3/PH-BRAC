# Data Description

This repository is being prepared for the PH-BRAC study. Anonymized feature-level data will be released after institutional review and before formal publication.

## Current Phase

No study data are included in the phase 1 repository setup.

## Planned Data Release

The planned release will include anonymized feature-level data and split files needed to reproduce the main analyses. The release will not include direct identifiers or raw source records.

Planned files may include:

| File | Description | Status |
| --- | --- | --- |
| `data/features.csv` | Anonymized feature-level table for model development and evaluation | Pending review |
| `data/data_dictionary.csv` | Variable names, descriptions, units, and allowed values | Pending review |
| `splits/train.csv` | Training split identifiers or row indices | Pending review |
| `splits/validation.csv` | Validation split identifiers or row indices | Pending review |
| `splits/test.csv` | Test split identifiers or row indices | Pending review |

## Privacy Notes

All released study files will be checked for de-identification before upload. Any additional access restrictions, if required by institutional review, will be documented before formal publication.
